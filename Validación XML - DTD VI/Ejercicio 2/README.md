# Ejercicio 2

## Descripción
Este archivo XML representa una película con un título opcional.

## Estructura
- **Elemento raíz**: `<pelicula>`  
  Representa una película.  
  - Atributos:
    - `titulo` (CDATA, opcional): Título de la película.

## Ejemplo
```xml
<pelicula titulo="La diligencia" />
```
